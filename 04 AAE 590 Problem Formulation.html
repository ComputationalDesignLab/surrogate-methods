This notebook supports material covered in the class for problem formulation. It contains two functions for demonstration. Please run the below block of code before you run any other block - it imports all the packages needed for this notebook.

import matplotlib.pyplot as plt
import numpy as np
import matplotlib

matplotlib.rcParams['contour.negative_linestyle'] = 'solid'

Example 1: Consider the following optimization problem:

\[\begin{split} \text{min } f(x_1,x_2) = x_1^2 - 2x_2^2 - 4x_1\\ \text{subject to } \begin{aligned} x_1 + x_2 \leq 6 \\ x_2 \leq 3 \\ x_1 \text{, } x_2 \geq 0 \end{aligned} \end{split}\]

Question 1: Transcribe the problem into the standard design optimization problem.

Answer: $\( \text{min } f(x_1,x_2) = x_1^2 - 2x_2^2 - 4x_1\\ \text{subject to } \begin{aligned} g_1(x_1,x_2) = x_1 + x_2 - 6 \leq 0 \\ g_2(x_1,x_2) = x_2 - 3 \leq 0 \\ g_3(x_1,x_2) = -x_1 \leq 0 \\ g_4(x_1,x_2) = -x_2 \leq 0 \end{aligned} \)$

Question 2: Plot the objective contours and constraints.

Answer: Following block of code defines a python function which return function and constraint values at given x.

def function_1(x1: np.ndarray, x2: np.ndarray):
    """
        Function to calculate the value of function at given x values.
    """
    
    f = x1**2 - 2*x2**2 - 4*x1
    g1 = x1 + x2 - 6
    g2 = x2 - 3
    g3 = -x1
    g4 = -x2
    
    return f, g1, g2, g3, g4

Following block of code uses the above defined function for plotting.

# Defining x and y values
x = np.linspace(-1,7,100)
y = np.linspace(-1,4,100)

# Creating a mesh at which values and 
# gradient will be evaluated and plotted
X, Y = np.meshgrid(x, y)

# Evaluating the value and gradient
Z, g1, g2, g3, g4 = function_1(X,Y)

# Plotting the graph

# Function contour
CS = ax.contour(X, Y, Z, levels=np.arange(-24, 5, 1), colors='b')

Orange thick lines: Shows infeasible side of the constraint.

Example 2: Plot the following objective function and constraints in the x – y plane:

\[\begin{split} f(x,y) = 2x + 3y - x^3 - 2y^2 \\ g_1(x,y) = x + 3y - 6 \leq 0 \\ g_2(x,y) = 5x + 2y - 10 \leq 0 \\ g_3(x,y) = -x \leq 0 \\ g_4(x,y) = -y \leq 0 \end{split}\]

Question: Identify the feasible region. By visual inspection, determine the minimum and maximum points for the objective function. For each of those points, give the objective function value and the active constraints (if any).

Answer: Following block of code defines a python function which return function and constraint values at given x.

def function_2(x: np.ndarray, y: np.ndarray):
    """
        Function to calculate the value of function at given x values.
    """
    
    f = 2*x + 3*y - x**3 - 2*y**2
    g1 = x + 3*y - 6
    g2 = 5*x + 2*y - 10
    g3 = -x
    g4 = -y
    
    return f, g1, g2, g3, g4

Quadrilateral ABCD is the feasible region. Since the function value increases as you move inwards from the boundary of the feasible region, four corners will be minimum.

Local minimum: \(A(0,0)\) with \(f(0,0) = 0\). Active constraints: \(g_3\) and \(g_4\)
Local minimum: \(B(0,2)\) with \(f(0,2) = -3\). Active constraints: \(g_1\) and \(g_3\)
Local minimum: \(C(1.39,1.54)\) with \(f(1.39,1.54) = 0\). Active constraints: \(g_1\) and \(g_2\)
Local minimum: \(D(2,0)\) with \(f(2,0) = -4\). Active constraints: \(g_2\) and \(g_4\)

Global minimum: \(D(2,0)\) with \(f(2,0) = -4\). Active constraints: \(g_2\) and \(g_4\)
